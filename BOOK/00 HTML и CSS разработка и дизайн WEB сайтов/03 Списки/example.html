Oмлeт

Блюда из яиц - мои любимые. Ниже я помещаю рецепт приготовления омлета.

Продукты

  • 2 яйца
  • 70 мл. молока
  • l/2 ч.л. соли
  • l ч.л. растительного масла

Приготовление

  1. Яйца перемешать с солью.
  2. Влить молоко , слегка перемешивая.
  3. Форму для запекания смазать растительным маслом.
  4. Вылить омлетную массу в смазанную форму. Поставить в предварительно разогретую до 250° С духовку и готовить омлет в духовке 20 - 30 минут.
  5. Bo время запекания на поверхности омлета образуется слегка румяная корочка, по степени зажаривания которой определите готовность омлета.
