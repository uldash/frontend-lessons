Напишите цикл for, который изменяет массив животных, делая их прекрасными! Например, если есть следующий массив:
var animals = ["Кот","Рыба","Лемур","Комодский варан",]
То наш цикл должен делать его таким:
var animals = ["Кот - прекрасное животное","Рыба - прекрасное животное - прекрасное животное","Лемур","Комодский варан - прекрасное животное",]
