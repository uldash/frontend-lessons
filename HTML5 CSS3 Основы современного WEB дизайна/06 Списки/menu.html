С помощью тега MENU в HTML документах реализуются списки меню. С помощью тега type можно задать тип меню:

  • context - контекстное меню
  • toolbar - меню в виде панели инструментов
  • list - меню - список
В html 4 тег не используется, в html 5 используется как контейнер для тега <command>
