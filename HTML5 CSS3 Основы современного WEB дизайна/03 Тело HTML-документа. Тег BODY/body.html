Атрибуты тега <BODY>

  • alink=color - цвет активной ссылки
  • link=color - цвет не просмотренной ссылки
  • vlink=color - цвет просмотренной ссылки
  • bgcolor=color - устанавливает фон документа
  • background=URL - указывает местоположение изображения, которое используется в качестве фонового
  • bgproperties - если установлен в положение fixed, то фоновое изображение не прокручивается, по умолчанию установлено положение nonfixed
  • text=color - цвет текста документа
  • leftmargin,rightmargin,topmargin=pixels - устанавливает размер полей документа
  • scroll - отображение полоы прокрутки документа, по умолчанию yes
