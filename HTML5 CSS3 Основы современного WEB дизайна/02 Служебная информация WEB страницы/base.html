Атрибут href=" " тега <BASE> в раздеде <HEAD> задает основной путь для относительных ссылок. Т.е. можно править документ на своем диске, а указать базовый путь для ссылок на работающий сайт, и тогда ссылки будут ссылаться на реальные страницы в интернете.

Ссылка на изображение Design.png

На странице может быть несколько тегов <BASE>, тег может располагаться в разделе <BODY>, область действия до конца документа или до следующего тега <BASE>.

Атрибут target=frame_name задает целевую ссылку на фрейм.
